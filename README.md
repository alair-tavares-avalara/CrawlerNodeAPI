## How to run the sample?

* Run from source using the following commands:
    * `npm install`
    * `npm start`

* Run tests
    * `npm test`

* Generate doc
    * `gulp doc`

* Run with Docker Compose
    * `docker-compose up`

    * http://localhost:5000/api/deputies
